# 第三周实验报告数据摘要

正式排版报告见 `reports/第三周实验报告.pdf`。PDF 由 `tools/build_week3_report.py` 根据仓库内 JSON/CSV 证据生成。

## 最终结果

- 稳定完成 40 轮训练。
- 严格分组 test：300 张图像、3472 个目标，mAP50=0.704，mAP50-95=0.403。
- 人工复核独立旧 test：150 张图像、414 个现有交通控制目标和 97 个漏标候选。
- 复核操作：删除 92、重分类 6、新增 36；复核后交通控制目标 358 个。
- 同一模型在复核前后总体 mAP50：0.640 -> 0.645；mAP50-95：0.469 -> 0.475。
- TensorRT GPU FP16：3.12 ms / 320.63 FPS；PyTorch GPU：5.18 ms / 192.98 FPS，加速 1.66 倍。
- 双模型融合：CPU 29.11 FPS，GPU 103.60 FPS。

## 结论边界

人工复核提高了测试标签及指标的可信度，但完整 1000 张数据集尚未全部替换为人工真值。推理速度来自 RTX 4070 Laptop GPU 笔记本软件管线，不等于真实汽车计算平台的摄像头到控制输出端到端延迟。
